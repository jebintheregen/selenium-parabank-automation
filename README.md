# 🚀 Selenium Automation Project - ParaBank

## 📌 Project Overview

This project demonstrates **automation testing of a banking web application** using Selenium WebDriver and TestNG.

The goal of this project is to automate real-world user workflows such as registration, login, fund transfer, and loan request.

---

## 🌐 Application Under Test

ParaBank Demo Website
👉 https://parabank.parasoft.com

---

## 🛠️ Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven
* ChromeDriver

---

## ✅ Test Scenarios Covered

* User Registration
* Login Functionality
* Open New Account
* Fund Transfer
* Loan Request

## ▶️ How to Run the Project

1. Clone the repository
2. Open in Eclipse or any IDE
3. Install Maven dependencies
4. Run `testng.xml` file
